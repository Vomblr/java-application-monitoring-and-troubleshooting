    HDD - % занятого места + кол-во операций чтение/запись в сек
    LAN - кол-во открытых соединений (% использования пула)
    RAM - % используемой памяти
    CPU - % использования
    
    Порты - список используемых и статус
    Class loading - кол-во загруженных классов по времени
    Garbage collector - кол-во освобожденной памяти за промежуток времени
    Profiling API - кол-во вызовов + время ожидания на конкретном методе
    
    TCP - кол-во открытых соединений
    Thread pools management - визуализировать треды: имена, среднее время блокировки, статусы
    JDBC pool - кол-во открытых соединений, частота, топ10 запросов
